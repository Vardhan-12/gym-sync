gym-sync
GymSync is a MERN-based gym coordination platform that enables users to plan workouts, monitor crowd density, and connect with compatible gym partners.

Author - Bala Vardhan Utla

GymSync is a smart gym coordination platform where users can:

Add planned workout time
View crowd density
Find matching gym partners
Send match requests
Chat after acceptance
Tech Stack:

MongoDB
Express
React
Node.js
